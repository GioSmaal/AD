Advanced Development Libraries

Favorieten:

1. AOS – Animate On Scroll
Deze library doet wat het zegt: Hij laat een animatie zien wanneer je scrolt, bijvoorbeeld voor websites met 1 lange pagina. Op hun eigen website https://michalsnik.github.io/aos/ staan al enorm veel voorbeelden weergeven. Ik denk dat dit enorm leuk staat voor een portfolio. Maar ook elke andere websitem om het meer levendig te maken.

2. Choreographer JS
De makers van deze library beweren css te kunnen animeren, en dat bleken geen leugens. Op de website https://christinecha.github.io/choreographer-js/ kreeg je een leuk voorbeeldje door te scrollen. Het deed me daarmee denken aan AOS, alleen hier was het scrollen niet door de pagina heen. Dit is vooral weer een library voor de design kant.

3. Cleave JS
Aan functionaliteit ook geen gebrek bij libraries. Cleave helpt je om cijfers naar het juiste format te krijgen. Het helpt om data te herkennen. Op de eigen website https://nosir.github.io/cleave.js/ zijn een aantal demo’s te vinden. Dit kan bruikbaar zijn voor invulvakken waar je informatie opvraagt. Bijvoorbeeld bij het maken van een account. Hier kun je een vak maken om je geboortedatum in te vullen en Cleave zorgt ervoor dat dit niet fout kan gaan. Het systeem geeft van te voren bijvoorbeeld “YYYY-MM-DD”(Year, month, day) aan.

Others:

4. Chart JS
Chart JS maakt het makkelijker om data om te zetten naar behulpzame grafieken. Dit kan voornamelijk handig zijn om onderzoeken en cijfers makkelijker inzichtelijk te maken op je website.
https://www.chartjs.org/

5. Granim JS
Granim maakt je achtergrond een stuk aantrekkelijker. Dit is een library voor design. Deze gebruikt een gradient in de achtergrond en geeft je zelfs de mogelijkheid om dingen te laten veranderen. Ik ben er geen groot fan van omdat het wat onrustig aanvoelt, maar het ziet er wel grappig uit.
https://sarcadass.github.io/granim.js/

6. Popper
Popper maakt tooltips of popovers makkelijk te gebruiken. Wanneer je een “dot” aantikt krijg je een animatie. Er zijn veel verschillende opties en het is zowel functionaliteit als design.
https://popper.js.org/

7. Premonish
Hoewel ik Premonish niet zo snel gebruikt zie worden in een van mijn eigen website, heb ik best veel plezier gehad om hun systeem op de eigen website te gebruiken. Het is een programma die voorspelt waar je muis heen gaat en zo al knoppen kan selecteren/oplichten voordat je er bent. Hier kan je misschien wel een leuk spel mee maken.
https://mathisonian.github.io/premonish/

8. Slick
Slick lijkt op een soort flexbox idee, maar dan met wat meer opties. Op de website laten ze wat mooie voorbeelden zien op het gebied van zowel design als funtionaliteit.
https://kenwheeler.github.io/slick/

9. TweenJS
Tween geeft net als wat andere libraries opties om te animeren. Hoewel ik deze niet zo nuttig vond als de anderen, ziet het er zeker leuk uit.
https://www.createjs.com/demos/tweenjs/tween_circles

10. Three JS
Three was een library die 3d models makkelijker maakt. Met Three zijn er veel games ontworpen, zelfs een aantal browser spellen die ik zelf gespeeld heb. Dit vond ik super interessant en op de website kun je zien wat er onder andere mee gemaakt is.
https://threejs.org/
